# 📖 Introduction
이 프로젝트에서는 축산업 종사자의 고령화 및 노동 인구 감소 해결을 위해 축산업 유지에 요구되는 인력을 최소화시키는 스마트 축산 시스템을 구현하고자 하였다. 프로젝트의 주요 목표는 자울주행 사료 급이 로봇, 축사 관리 UI, IoT와 딥러닝 기술을 이용한 축사 및 가축 관리다.
